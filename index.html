A bit about me

I'm passionate about leveraging my technical background, interdisciplinary lenses, and thought-provoking questions to build technologies and contribute to teams that are rooted in empowerment, social equity, and intentional impact.

I graduated from the University of Washington with a degree in Computer Science and minor in Education, Learning, and Society.

Projects

Here are a few tech projects I've worked on.

Allyship App

Cross-platform mobile Allyship App from Geeking Out Kids of Color (GOKiC) to equip people with daily reflections, weekly articles, and calls to action for learning and mobilization around intersectional allyship. Using Ionic framework.

Nonprofit Website

React web app for Geeking Out Kids of Color (GOKiC), a community organization rooted in decolonized tech education. Hosted by Azure, integrated with Google Analytics.

Guessing Game

Jeopardy-esque guessing game to review key terms from Subtractive Schooling: U.S.-Mexican Youth and the Politics of Caring (Angela Valenzuela, 1999). Using Python on Repl platform.

WOC Exploratory Game

Interactive exploratory game to learn virtually about womxn of color organizers and advocates for Geeking Out Kids of Color (GOKiC) youth program. Using Lua on Roblox Studio platform.

Website Templates

Simple website templates for Geeking Out Kids of Color (GOKiC) youth program. Using HTML, CSS, JS on Repl platform.

Word Suggestor

Chat application back-end to take Facebook Messenger chat history JSON to build word suggestion feature using Java.

Flights App

Flight-booking command line interface for users to login, search, book, and pay for flights using Java and SQL.

Campus Landmark Hunt App

Landmark hunt mobile application for users to select landmark photo of University of Washington campus and be guided there. Using Java, XML, and Google Maps API on Android Studio.

Thoughts

I like writing about my thoughts. I share some of them on Instagram. Here are a few snippets.

"Exotic"

... dragon fruit / thanh long, tou liang / sầu riêng / lau lien / durian, chôm chôm / rambutan, longan, papaya / đu đủ. These fruits taste like wholeness. So what does it mean when some people refer to them as "exotic"?

Stereotypes & Expectations

Try asking the kiddos in your life what they think about genders and colors. Explore the notions of stereotypes and expectations. How can you co-develop questions and understanding?

Weekends, Biases, & Hobbies
